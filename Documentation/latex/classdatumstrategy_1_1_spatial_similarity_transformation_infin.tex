\doxysubsection{Detailed Description}
Algorithmic realization of the 3D similarity transformation also known as 7-\/parameter transformation. 

The method takes all needed parameters from the class Control\+Parms that extends the class Ellipsoid\+Parms. Needed parameter for the 7-\/Param-\/\+Transformation\+: 3 translation-\/, 1 scale-\/ and 3 rotation -\/parameters. The start coordinates are taken from the class Geographic\+Coordinate and have to be converted into cartesian coordinates to realize the transformation. The 7-\/Param-\/\+Transformation is done with cartesian coordinates. After the \mbox{\hyperlink{class_transformation}{Transformation}}, the new cartesian coordinates are converted into ellipsoidal coordinates by an iterative process.
