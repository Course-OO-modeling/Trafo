\doxysubsection{Ausführliche Beschreibung}
Algorithmic realization of the Euler\textquotesingle{}s transformation. 

The method takes all needed parameters from the class Control\+Parms that extends the class Ellipsoid\+Parms. Needed parameter for the \mbox{\hyperlink{classdatumstrategy_1_1_eulers_transformation}{Eulers\+Transformation}}\+: 3 translation-\/, 1 scale-\/ and 3 rotation -\/parameters. The start coordinates are taken from the class Geographic\+Coordinate and have to be converted into cartesian coordinates to realize the transformation. The Translation is done with cartesian coordinates and the rotation is implemented with trigonometric function. After the \mbox{\hyperlink{class_transformation}{Transformation}}, the new cartesian coordinates are converted into ellipsoidal coordinates by an iterative process.
